# AI Assistance Policy

This project permits the use of AI-assisted tools as part of the development workflow, provided that all contributions remain legally, technically, and ethically maintainable.

AI tools may be used for activities such as brainstorming, code review assistance, refactoring suggestions, documentation drafting, test generation, and exploratory implementation support.

However, AI-assisted output is not accepted blindly. Every contribution must be reviewed, understood, and accepted by a human contributor or maintainer. The person submitting the contribution is responsible for ensuring that the submitted work is correct, maintainable, and compatible with this project's license.

## Contribution Requirements

By submitting a contribution to this project, contributors confirm that:

- They have the right to submit the contribution under this project's license.
- The contribution does not knowingly contain code copied from proprietary, leaked, confidential, or otherwise unauthorized sources.
- The contribution does not knowingly include material with a license that is incompatible with this project.
- Any use of AI tools was reviewed and validated by the contributor before submission.
- The contributor is willing to maintain or explain the submitted work if requested by the maintainers.

## Maintainer Discretion

The maintainers reserve the right to reject, request changes to, or remove any contribution if there are concerns about originality, licensing compatibility, maintainability, security, or code quality.

AI-assisted contributions are evaluated using the same standards as any other contribution.

## Third-Party Material

Any third-party code, assets, documentation, algorithms, or significant external references used in this project must be clearly identified and documented where appropriate.

When required, third-party notices should be added to the relevant notice or attribution files.

## Project Position

This policy does not prohibit AI-assisted development. Its purpose is to ensure that the project remains transparent, legally responsible, and maintainable over time.
